## 📌 О проекте

**AXIS** — голосовой ассистент, работающий полностью локально, без отправки данных в облако. Он понимает речь, выполняет команды на компьютере, отвечает на вопросы и запоминает информацию о пользователе.

Проект создаётся в учебных целях и как личный инструмент для автоматизации повседневных задач.

## 🚀 Возможности (реализовано)

| Функция | Описание |
|---------|----------|
| 🎤 **Голосовой ввод** | Распознавание речи через Vosk (офлайн) |
| 🗣️ **Голосовой вывод** | Синтез речи через Edge-TTS |
| 🧠 **Локальная LLM** | Llama 3.1 8B / Gemma 3 4B через Ollama |
| 💾 **Долговременная память** | Запоминает факты о пользователе (имя, предпочтения) |
| 🖥️ **Управление ПК** | Открытие программ, сайтов, таймеры |
| 💬 **Текстовый чат** | Графический интерфейс на PySide6 |

## 🛠️ Технологии

| Компонент | Технология |
|-----------|------------|
| **Язык** | Python 3.10+ |
| **LLM** | Ollama + Llama 3.1 8B |
| **STT** | Vosk (русская модель) |
| **TTS** | Edge-TTS |
| **GUI** | PySide6 (Qt for Python) |
| **Архитектура** | Модульная, с QThread для асинхронности |
