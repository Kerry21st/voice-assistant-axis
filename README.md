## О проекте

**AXIS** — голосовой ассистент, работающий полностью локально, без отправки данных в облако. Он понимает речь, выполняет команды на компьютере, отвечает на вопросы и запоминает информацию о пользователе.

Проект создаётся в учебных целях и как личный инструмент для автоматизации повседневных задач.

## Возможности (реализовано)

| Функция | Описание |
|---------|----------|
| **Голосовой ввод** | Распознавание речи через Vosk (офлайн) |
| **Голосовой вывод** | Синтез речи через Edge-TTS |
| **Локальная LLM** | Llama 3.1 8B / Gemma 3 4B через Ollama |
| **Долговременная память** | Запоминает факты о пользователе (имя, предпочтения) |
| **Управление ПК** | Открытие программ, сайтов, таймеры |
| **Текстовый чат** | Графический интерфейс на PySide6 |

## Установка и запуск

### Требования
- Windows 10/11
- Python 3.10 или выше
- 8+ ГБ ОЗУ (рекомендуется 16 ГБ)

### Шаг 1. Клонирование
```
git clone https://github.com/Kerry21st/voice-assistant-axis.git
cd voice-assistant-axis
```
Шаг 2. Виртуальное окружение
```
python -m venv venv
venv\Scripts\activate
```
Шаг 3. Установка зависимостей
```
pip install -r requirements.txt
```
Шаг 4. Установка Ollama и модели
# Скачай Ollama с https://ollama.com/
```
ollama pull llama3.1:8b
```
Шаг 5. Скачивание модели Vosk
Скачай vosk-model-small-ru-0.22 с сайта Vosk

Распакуй в папку models/vosk-model-small-ru-0.22

Шаг 6. Запуск
```
python run_assistant.py
```

## Примеры использования
| Команда | Результат |
|---------|----------|
| `"Открой калькулятор"` | Запускается калькулятор Windows |
| `"Открой Яндекс Музыку"` | Открывается сайт (или программа, если установлена) |
| `"Сколько времени?"` | Ассистент называет текущее время |
| `"Меня зовут Дима"` | Запоминает имя |
| `"Как меня зовут?"` | Отвечает "Дима" |
| `"Что я люблю?"` | Отвечает из сохранённых фактов |
| `"Поставь таймер на 5 минут"` | Запускает таймер с уведомлением |
| `"Открой мой сайт"` | Открывает fieldmate.tw1.ru в браузере |
| `"Кто ты?"` | Представляется как AXIS |
| `"Спасибо"` | Отвечает "Всегда рад помочь" |

## Архитектура

```
[Голос пользователя] → [Vosk (STT)] → [Текст]
                                    ↓
[Текстовая команда] → [Обработка] → [Выбор модели]
                                    ↓
[Llama 3.1 / Gemma 3] ← [Контекст + память]
                                    ↓
[System Commands] ← [Function calling]
                                    ↓
[Ответ] → [Edge-TTS (TTS)] → [Голос ассистента]
```
